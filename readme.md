# GRS Vegas Slots (v2.0)

🎰 **Новые фичи:**
1. **1 попытка + 5 мин КД** (после попытки добавляется новая).
2. **Авторизация** (никнейм сохраняется в браузере).
3. **Live-лента выигрышей** (реальное время, без перезагрузки).

---

## **Как играть:**
1. Введите никнейм и нажмите **"Войти"**.
2. Нажмите **"СПИН!"** для вращения барабанов.
3. Если выпадут одинаковые призы — вы выиграли!
4. После спина **5 минут КД** (кнопка блокируется).
5. Через 5 минут **возвращается новая попытка**.

---

## **Live-лента**
- Все выигрыши отображаются в реальном времени.
- Сообщения появляются вверху и "прокручиваются".

---

## **Технические детали**
- **Хранение данных**: `localStorage` (сохраняется в браузере).
- **Развёртывание**: GitHub Pages (бесплатно).
